1. Import :
   ```python

   from src.models import Track, SyncOperation
   ```

2. Créer les données :
   ```python
   missing_track = Track(
       urlmd5="abc123",
       title="Song Title",
       artist="Artist Name",
       # ... autres champs
   )
   
   suggestions = [
       (track1, 95.0),  # 95% de correspondance
       (track2, 68.0),
       (track3, 45.0),
   ]
   ```

3. Définir les callbacks :
   ```python
   def handle_apply(operation: SyncOperation) -> bool:
       # Exécuter l'opération
       try:
           db.execute_sync(operation)
           return True
       except Exception as e:
           print(f"Error: {e}")
           return False
   
   def handle_next():
       # Traiter le morceau suivant
       next_track = get_next_track()
       # ... ouvrir dialogue pour next_track
   ```

4. Ouvrir le dialogue :
   ```python
   show_match_dialog(
       parent=main_window,
       missing_track=missing_track,
       suggested_matches=suggestions,
       on_apply=handle_apply,
       on_next=handle_next
   )
   ```

INTERACTIONS
============

✓ Sélectionner une suggestion → mise à jour SQL en temps réel
✓ Changer l'action (COPY/MERGE) → mise à jour SQL
✓ Éditer le playcount → mise à jour SQL
✓ Cliquer "Appliquer" → validation → callback on_apply() → ferme si OK
✓ Cliquer "Ignorer" → callback on_next() → ferme
✓ Cliquer "Annuler" → ferme sans action


CODE COULEUR
============

✓ 95% (vert) → Correspondance excellente
⚠ 68% (orange) → Correspondance acceptable  
✗ 45% (rouge) → Correspondance faible

Score < 60% → Demande confirmation avant apply


CALLBACKS
=========

on_apply(operation: SyncOperation) -> bool :
  - Appelé quand l'utilisateur clique "Appliquer"
  - Doit retourner True si succès, False si erreur
  - Si True → dialogue se ferme
  - Si False → dialogue reste ouvert + message d'erreur

on_next() -> None :
  - Appelé quand l'utilisateur clique "Ignorer"
  - Permet traiter le morceau suivant (batch)
  - Optionnel


VALIDATION
==========

✓ Au moins un match sélectionné
✓ Playcount >= 0
✓ Confirmation si score < 60%
✓ Messages d'erreur clairs


WORKFLOW COMPLET

# Détection
detector = SyncDetector(db)
missing_tracks = detector.find_missing_in_alternative()

# Pour chaque morceau manquant
for missing in missing_tracks:
    
    # Matching
    matcher = TrackMatcher()
    suggestions = matcher.find_best_matches(missing)
    
    # Afficher dialogue
    def apply(op):
        db.execute_sync(op)
        main_window.refresh()
        return True
    
    def next():
        # passer au suivant
        pass
    
    show_match_dialog(
        main_window,
        missing,
        suggestions,
        on_apply=apply,
        on_next=next
    )

Q: Comment ouvrir le dialogue ?
