🔌 API
=======

Fonction Helper :
  show_match_dialog(
      parent: tk.Widget,
      missing_track: Track,
      suggested_matches: List[tuple[Track, float]],
      on_apply: Callable[[SyncOperation], bool] = None,
      on_next: Callable = None
  ) -> None

Callbacks :
  on_apply(operation: SyncOperation) -> bool
    - True = succès, ferme le dialogue
    - False = erreur, reste ouvert
    
  on_next()
    - Appelé quand "Ignorer" cliqué
    - Support mode batch

🏗️ ARCHITECTURE
================

MatchDialog (tk.Toplevel)
├── Section 1 : Morceau manquant (Label widgets)
├── Section 2 : Suggestions (Canvas + Scrollbar + Radio buttons)
├── Section 3 : Action (Radio buttons + Spinbox + Checkbox)
├── Section 4 : SQL (Text read-only)
└── Section 5 : Boutons (3 buttons)

Interactions :
  - Sélectionner suggestion → _on_suggestion_selected() → _update_sql_preview()
  - Changer action → _on_action_changed() → _update_sql_preview()
  - Éditer spinbox → automatique _update_sql_preview()
  - Cliquer "Appliquer" → _on_apply_click() → validation → callback
  - Cliquer "Ignorer" → _on_ignore_click() → callback → fermeture
  - Cliquer "Annuler" → destroy()
