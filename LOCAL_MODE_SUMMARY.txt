✨ AVANTAGES MODE LOCAL

  ✅ Pas de Docker requis
  ✅ Démarrage < 2 secondes (vs 10-15 sec Docker)
  ✅ GUI native Tkinter
  ✅ Dépendances auto-installées
  ✅ Parfait pour développement
  ✅ Moins de ressources système

═════════════════════════════════════════════════════════════════════════════

🔧 EXEMPLES COMPLETOS

# DOCKER MODE
python3 scripts/launch.py /volume1/docker/squeezebox-lms/prefs
python3 scripts/launch.py /var/lib/squeezeboxserver/prefs
python3 scripts/launch.py ~/Library/Application\ Support/Squeezebox/prefs

# LOCAL MODE
python3 scripts/launch.py /volume1/docker/squeezebox-lms/prefs --local
python3 scripts/launch.py /var/lib/squeezeboxserver/prefs --local
python3 scripts/launch.py ~/Library/Application\ Support/Squeezebox/prefs --local

═════════════════════════════════════════════════════════════════════════════

🎮 OPTIONS LOCAL

  Démarrer              : python3 scripts/launch.py /path --local
  Voir logs             : python3 scripts/launch.py /path --local --logs
  Logs en temps réel    : python3 scripts/launch.py /path --local --logs --follow
  Vérifier statut       : python3 scripts/launch.py /path --local --status
  Mode verbose (debug)  : python3 scripts/launch.py /path --local --verbose

📋 COMPARAISON DOCKER VS LOCAL

  Aspect               │ Docker              │ Local Python
  ─────────────────────┼─────────────────────┼──────────────────
  Interface            │ VNC (web + client)  │ Tkinter native
  Démarrage            │ 10-15 sec           │ < 2 sec
  Dépendances          │ Docker requis       │ Python requis
  Ressources           │ Plus lourd          │ Plus léger
  Développement        │ Moins idéal         │ Parfait ✅
  Production           │ Idéal ✅            │ Acceptable
  Accès à GUI          │ VNC (réseau)        │ Directement
  Isolation            │ Complète            │ Système partagé
